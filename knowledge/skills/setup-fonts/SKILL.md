# Setup Fonts

## Phase 1 — Audit

### 1.1 Extract needed fonts from design-system.json

Read `.wpaikit/design-system.json` → `typography` section.

Parse each key (`FontFamily/sizePx/weight`) to extract:
- Font family name (e.g. `Inter`)
- All weights used (e.g. `400`, `600`, `700`)

Build `NEEDED_FONTS`:
```
Inter: [400, 600, 700]
```

Skip deprecated typography entries.

### 1.2 Scan font files

Scan `frontend/src/fonts/` recursively. Collect all files with extensions:
`.ttf`, `.otf`, `.woff`, `.woff2`

For each file, parse:
- **Family** — from filename (e.g. `Inter-Bold.ttf` → `Inter`)
- **Weight** — from filename keyword:

| Keyword | Weight |
|---|---|
| `Thin` | 100 |
| `ExtraLight`, `UltraLight` | 200 |
| `Light` | 300 |
| `Regular`, absence of keyword | 400 |
| `Medium` | 500 |
| `SemiBold`, `DemiBold` | 600 |
| `Bold` | 700 |
| `ExtraBold`, `UltraBold` | 800 |
| `Black`, `Heavy` | 900 |

- **Style** — `Italic` in filename → `italic`, otherwise `normal`
- **Format** — file extension

Build `FOUND_FONTS`:
```
Inter-Regular.ttf     → Inter, 400, normal, ttf
Inter-Bold.ttf        → Inter, 700, normal, ttf
Inter-SemiBold.woff2  → Inter, 600, normal, woff2
Roboto-Regular.ttf    → Roboto, 400, normal, ttf
```

### 1.3 Show audit table

```
=== Font Audit ===

Needed (from design-system.json):
  Inter  400, 600, 700

Found in frontend/src/fonts/:
  ✓ Inter-Regular.ttf    (400, normal)
  ✓ Inter-SemiBold.ttf   (600, normal)
  ✓ Inter-Bold.ttf       (700, normal)
  ✗ Roboto-Regular.ttf   (not in design system → will be removed)

Missing:
  (none)
```

If any needed font weight is missing from `frontend/src/fonts/`:
```
  ⚠ Inter 500 — not found. Add the font file before proceeding.
```

Stop and warn if a critical font is missing. User can continue anyway or cancel.

## Phase 2 — Clean

Show files to delete:
```
Files to remove:
  frontend/src/fonts/Roboto-Regular.ttf
  frontend/src/fonts/Roboto-Bold.woff2

Remove these files? [yes / cancel]
```

Wait for confirmation. On `yes` → delete listed files.

Never delete:
- Files that match a needed font family + weight
- Files that were already converted (woff/woff2 of a kept font)

## Phase 3 — Convert

For each kept font file, determine what conversions are needed:

| Source format | Generate |
|---|---|
| `.ttf` or `.otf` | woff2 + woff |
| `.woff` | woff2 only |
| `.woff2` | nothing (already optimal) |

Skip conversion if the target file already exists.

### Conversion commands

Check that `fonttools` and `brotli` are available:
```bash
python3 -c "import fontTools; import brotli" 2>/dev/null
```

If not installed → stop: "Run `pip install fonttools brotli` and try again."

**TTF/OTF → woff2:**
```bash
python3 -c "
from fontTools.ttLib import TTFont
font = TTFont('INPUT_PATH')
font.flavor = 'woff2'
font.save('OUTPUT_PATH')
"
```

**TTF/OTF → woff:**
```bash
python3 -c "
from fontTools.ttLib import TTFont
font = TTFont('INPUT_PATH')
font.flavor = 'woff'
font.save('OUTPUT_PATH')
"
```

**woff → woff2:**
```bash
python3 -c "
from fontTools.ttLib import TTFont
font = TTFont('INPUT_PATH')
font.flavor = 'woff2'
font.save('OUTPUT_PATH')
"
```

Output path = same directory as input, same filename, different extension.

Example:
- `Inter-Regular.ttf` → `Inter-Regular.woff2` + `Inter-Regular.woff`
- `Inter-SemiBold.woff` → `Inter-SemiBold.woff2`

Show progress per file:
```
Converting:
  Inter-Regular.ttf  →  Inter-Regular.woff2 ✓
  Inter-Regular.ttf  →  Inter-Regular.woff  ✓
  Inter-Bold.ttf     →  Inter-Bold.woff2    ✓
  Inter-Bold.ttf     →  Inter-Bold.woff     ✓
```

## Phase 4 — Generate @font-face

### 4.1 Write `frontend/src/fonts/_fonts.scss`

One `@font-face` block per font family + weight + style combination.
Use woff2 as primary, woff as fallback. Never reference ttf/otf in CSS.

```scss
// Fonts
// Generated by /setup-fonts

@font-face {
  font-family: 'Inter';
  src: url('../fonts/Inter-Regular.woff2') format('woff2'),
       url('../fonts/Inter-Regular.woff') format('woff');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'Inter';
  src: url('../fonts/Inter-SemiBold.woff2') format('woff2'),
       url('../fonts/Inter-SemiBold.woff') format('woff');
  font-weight: 600;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'Inter';
  src: url('../fonts/Inter-Bold.woff2') format('woff2'),
       url('../fonts/Inter-Bold.woff') format('woff');
  font-weight: 700;
  font-style: normal;
  font-display: swap;
}
```

Rules:
- Always `font-display: swap`
- URL path relative to the SCSS file location (`../fonts/`)
- Only include weights that exist after Phase 3
- If only woff2 exists (no woff fallback) → use woff2 only

### 4.2 Update main.scss

Find the main SCSS entry file (`frontend/src/main.scss`, `index.scss`, or `app.scss`).

Add `@import` at the very top (fonts must load before everything else):

```scss
@import 'fonts/fonts';
```

If the import already exists → skip.

## Boundaries

- Never delete font files without confirmation
- Never delete woff/woff2 files generated in Phase 3
- Never reference ttf/otf in generated CSS
- Always use `font-display: swap`
- Do not modify `design-system.json`
